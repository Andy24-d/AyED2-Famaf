# AyED2-Famaf

Repositorio de trabajo para la materia **Algoritmos y Estructuras de Datos 2 (FaMAF)**.

## Organización inicial

La cursada se organiza en dos modalidades principales:

- `teorico-practico/`: resolución de contenidos teóricos y prácticos en el lenguaje algorítmico utilizado en la materia.
- `laboratorio/`: desarrollo de código en **C** para las actividades de laboratorio.
